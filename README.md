## Setup

```
cd toy-erc20
yarn
```

## Usage

### Compile

```
yarn hardhat compile
```

### Deploy

```
yarn hardhat deploy
```

### Testing

```
yarn hardhat test
```

### Test Coverage

```
yarn hardhat coverage
```
